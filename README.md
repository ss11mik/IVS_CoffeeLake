Prostředí
---------

- multiplatformní, běží v Javě
- požadována Java rutime (JRE) na klientském PC (vyvinuto a testováno na verzi 11.0.6)

Požadavky pro kompilaci

- Gradle (5.2.1)
- JRE (11)
- GNU make
- pdflatex (pro vygenerování manuálu, pdf dokumentace a profiling reportu)
